## Descripción

Una aplicación web para verificar y validar datos de usuarios mediante **expresiones regulares (REGEX)**. 

Proyecto desarrollado para la asignatura de **Lenguajes y Autómatas** utilizando tecnologías web modernas sin dependencias externas.

> Toda la lógica está desarrollada con **HTML puro**, **CSS** y **Vanilla JavaScript** para garantizar máxima compatibilidad y rendimiento.
